# CS-300 Portfolio

## Reflection

In this course, the main problem I worked on was organizing and managing course data efficiently using different data structures. The goal was to read course information from a file and allow users to search, sort, and display courses in a structured way.

To approach this problem, I used data structures like binary search trees to store and organize the data. This helped improve efficiency when searching and printing course information. Understanding how different data structures impact performance was very important in completing these projects.

While working on the assignments, I ran into challenges such as file input errors and debugging issues when the program could not locate the correct file. I overcame these challenges by carefully checking file paths, adjusting my code, and testing step by step until everything worked correctly.

This project helped expand my understanding of how to design programs using structured approaches. I learned how to break problems into smaller steps and apply the correct data structures to solve them efficiently.

Overall, this experience improved how I write code by making it more organized, readable, and adaptable. I now have a better understanding of how to build programs that are easier to maintain and scale in the future.
